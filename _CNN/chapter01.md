# 一、主題
利用MatLab中的深度網路設計器(Deep Network Designer)訓練電腦對地理環境的辨別，包含沙漠、冰原、雨林和莽原等不同景觀。再透過上傳照片至訓練好的模型來測試準確率，及不同參數下對訓練模型的影響。而訓練中使用了卷積神經網路(CNN)是一種專門處理圖像、影片等網格化數據的深度學習模型，模仿人類視覺系統自動學習和提取圖像的層次化特徵，並進行分類或預測。
