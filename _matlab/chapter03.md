# 主題三：Matlab_握把式圖形與GUI設計

## 大綱

本單元主要以基本的 MATLAB 程式設計為主，如握把式圖形(HandleGraphics)，以及相關的圖形使用者介面(GUI，Graphic User Interface)設計，而利用握把式圖形以及 GUI 設計技能達到更改 MATLAB 產生的圖形的效果。
